Introduction

Hello, my name is Ryan McGuinn. I'm a Web Developer and Designer based out of Seattle, WA. I strive to create applications that are equally beautiful as they are efficient.

Technology has always been my passion and since I discovered programming, the feeling of creating something from scratch has become quite addictive. I am always putting in the work to not only gain new skills, but to improve upon my existing ones.

For me, programming is just plain exciting. I look forward to bringing this passion and enthusiasm with me, wether that be for a great company, or to my clients as a freelancer.

Design

I start off gathering inspiration from relevant images and choose a color palette to compliment the design. Using the Adobe Creative Suite, I can build projects that look great and provide the user with a great experience.

Code

Then on to the programming. I'll take the design and put it up on one monitor, while using the other to create a pixel perfect reproduction using a mixture of JavaScript, SASS/CSS, HTML and whatever else is needed

Test

Lastly, it's on to the testing phase. Here is where I'll take the foundation I've layed and perfect it. This includes cross browser testing, debugging, increasing page load times and optimizing it all for a great UX.

Responsive Design

I build websites to be responsive, meaning the design changes to look great no matter what device you're using. From large desktop computers, to small smartphones and everything in between.
